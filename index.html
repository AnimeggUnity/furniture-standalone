安裝教學

簡單三個步驟，開啟高效管理體驗。

1

下載專案源碼

從 GitHub 下載本專案並解壓縮至你的電腦資料夾中。

2

進入開發者模式

開啟 Chrome 瀏覽器，在網址列輸入 chrome://extensions/ 並開啟右上角的「開發者模式」。

3

載入擴充功能

點擊「載入未封裝擴充功能」，選取剛才解壓縮的專案資料夾即可完成安裝。
